An interactive board state live calculator for the Grinchmas game that helps players quickly compile the total amount of points they gathered in a round.

## Features

- **Live Calculation**: Your score updates automatically as you enter values
- **Base Points**: Enter your base points from the round
- **Multiple Bonuses**: Support for various bonus types:
  - Present Collection Bonus
  - Ornament Bonus
  - Stocking Bonus
  - Candy Cane Bonus
  - Grinch Challenge Bonus
  - Whoville Bonus
- **Score Multipliers**: Apply multipliers (1x, 1.5x, 2x, 3x) to your total score
- **Reset Functionality**: Quickly clear all values to start a new calculation
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## How to Use

1. Open `index.html` in any modern web browser
2. Enter your base points from the round
3. Add any bonuses you've earned
4. Select a multiplier if applicable
5. Your total score is calculated and displayed automatically
6. Click "Reset All" to start a new calculation

## Files

- `index.html` - Main calculator interface
- `calculator.js` - Score calculation logic
- `styles.css` - Festive styling and layout

## Local Setup

Simply open the `index.html` file in your web browser - no build process or dependencies required!

```bash
# Clone the repository
git clone https://github.com/72Rings/Grinchmas_calc.git

# Open in browser
cd Grinchmas_calc
open index.html  # macOS
# or
start index.html  # Windows
# or
xdg-open index.html  # Linux
```

## Browser Support

Works in all modern browsers including Chrome, Firefox, Safari, and Edge.
